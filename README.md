## ☁︎ 소개
- 멋쟁이사자처럼에서 주최한 WEB 캐릭터 디자인 경진대회의 참여작입니다.

<br/>

## 👩‍💻 구현 방식
- CSS `animation`, `@keyframes`을 적극 활용하여 구현했습니다.
- `position` 으로 위치를 지정하여 세밀화 하였습니다.
